# Welcome to MDX

This is an MDX page. You can write **Markdown** and use React components directly in your content.

## Features

- Write content in Markdown
- Import and use React components
- Full TypeScript support

```tsx
import { Button } from '@/components/ui/button'

export default function Page() {
  return <Button>Click me</Button>
}
```

## Try it out

Run `pnpm dev` and navigate to `/docs` to see this page.
